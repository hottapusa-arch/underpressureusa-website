# Under Pressure USA Website v1.0

This release stabilizes the current website foundation.

Included fixes:
- Home hero no longer hides under the blurred header
- Partner hero has its own independent layout
- Old stacked Partner-page CSS patches cleaned out
- Spotify / YouTube Listen & Watch section preserved
- Media Kit button points to the PDF
- Mobile spacing improved

Install:
1. Unzip this package.
2. Copy the contents into your local `underpressureusa-website` folder.
3. Replace existing files when prompted.
4. Commit summary: `Release website v1.0`
5. Push origin.

After deploy, verify:
- Home hero title is visible
- Partner hero title is visible
- Spotify button opens Spotify
- YouTube button opens YouTube
- Media Kit button opens PDF
